# 변수 Variable

### 데이터의 저장단위

- 비트 Bit
    - 컴퓨터가 나타내는 저장 최소 단위
    - 2진수 값 하나를 저장
- 바이트 Byte
    - 데이터 처리 또는 문자의 초소 단위
    - 8 bits

### 변수의 정의 및 종류

- 변수의 정의
    - 메모리 공간 RAM 에 하나의 값을 기록하기 위한 공간
- 변수의 자료형 Type
    - 기본형 Primitive Type
        - 실제 데이터 값을 저장
        - 각 자료형 별 크기가 다름
    - 참조형 Reference Type
        - 데이터가 저장되어 있는 주소(객체의 주소)를 저장
        - 기본형을 제외한 나머지(String 등), 사용자 정의 자료형
        - 4Byte

### 자료형 종류
- 논리
  - boolean, 1byte
- 문자
  - 문자
    - char, 2byte
  - 문자열
    - String, reference type
- 숫자
  - 정수형
    - byte, 1byte
    - short, 2byte
    - int, 4byte
    - long, 8byte
  - 실수형
    - float, 4byte
    - double, 8byte

### 변수의 명명 규칙

1. 대소문자 구분이 없다
2. 길이 제한이 없다
3. 예약어를 사용할 수 없다
4. 숫자로 시작할 수 없다
5. 특수문자는 '_'와 '$' 외 사용할 수 없다
6. 여러 단어의 이름은 단어의 첫 글자를 대문자로 한다.

### 상수

- 상수의 정의
    - 한번만 저장 가능한 공간
    - 초기화 이후 다른 데이터(값) 대입 불가능
- 상수 선언의 예
    
    ```java
    final int age;
    final double PI = 3.14;
    ```
    

# 형변환 Casting

### 컴퓨터의 값 처리 원칙

- 같은 종류 자료형만 대입 가능
- 같은 종류 자료형만 계산 가능
- 계산 결과는 같은 종류의 값이 나와야 함

### 형변환의 종류

- 자동 형변환
    - 연산 시 컴파일러가 자동으로 형 변환을 수행
    - byte → short → int → long → float → double
    - char → int → ...
    - byte와 short 연산
    
    <aside>
    💡 Java에서는 byte와 short 연산 시 자동으로 결과 값을 int형으로 반환
    
    </aside>
    
- 강제 형변환
    - 데이터가 큰 자료형에서 작은 자료형으로 변경 시 수행
    - 데이터의 손실이 발생할 수 있음

# 변수와 메모리 구조

- Static
    - 예약어로 선정된 필드
    - 메소드가 저장되는 공간
    - 클래스 변수 등
- Heap
    - new 연산자에 의해 동적으로 할당하고 저장되는 공간
    - 객체, 배열등
- Stack
    - 메소드를 호출하면 자동 생성
    - 메소드가 끝나면 자동 소멸
    - 지역 변수, 매개변수 parameter, 메소드 호출 스택 등

# 실습 예제

### 변수 실습

- 문제
    
    [변수 연습문제]
    => 패키지 생성 : com.example
    => 실행용 클래스 생성 : TestMain.java
    >> main() 메소드 포함 : 각 문제별 메서드 실행 테스트함
    => 기능제공용 클래스 생성 : Example.java
    
    [연습 1]
    => 메소드명 : public void example1(){}
    => 실행내용 : 키보드로 정수 두 개를 입력받아, 두 수의 합, 차, 곱
    나누기한 몫과 나머지 출력
    => 입력예
    첫번째 정수 : 23
    두번째 정수 : 7
    => 출력예		  계산식
    더하기 결과 : 30  --> (값1 + 값2)
    빼기 결과 : 16   --> (값1 - 값2)
    곱하기 결과 : 161 --> (값1 * 값2)
    나누기한 몫 : 3   --> (값1 / 값2)
    나누기한 나머지 : 2  --> (값1 % 값2)
    
    [연습 2]
    => 메소드명 : public void example2(){}
    => 실행내용 : 키보드로 가로, 세로 값을 실수형으로 입력받아
    사각형의 면적과 둘레를 계산하여 출력
    계산공식 - 면적 : 가로 * 세로
    둘레 : (가로 + 세로) * 2
    => 입력예
    가로 : 13.5
    세로 : 41.7
    => 출력예
    면적 : 562.95
    둘레 : 110.4
    
    [연습 3]
    => 메소드명 : public void example3(){}
    => 실행내용 : 영어 문자열 값을 키보드로 입력받아
    각 자리의 문자를 출력
    => 입력예
    문자열을 입력하시오 : apple
    => 출력예 (3개만 출력해 봄)
    첫번째 문자 : a
    두번째 문자 : p
    세번째 문자 : p
    

```java
package com.example;

import java.util.Scanner;

public class Example {
	public void example1(){
		Scanner sc = new Scanner(System.in);
		
		System.out.println("첫번째 정수 : ");
		int a = sc.nextInt();
		System.out.println("두번째 정수 : ");
		int b = sc.nextInt();

		System.out.println("더하기 결과 : " + (a+b) + " ---> (값1 + 값2)");
		System.out.println("빼기 결과 : " + (a-b) + " ---> (값1 - 값2)");
		System.out.println("곱하기 결과 : " + (a*b) + " ---> (값1 * 값2)");
		System.out.println("나누기한 몫 : " + (a/b) + " ---> (값1 / 값2)");
		System.out.println("나누기한 나머지 : " + (a%b) + " ---> (값1 % 값2)");
	}
	public void example2() {
		Scanner sc = new Scanner(System.in);

		System.out.println("가로 : ");
		double a = sc.nextDouble();
		System.out.println("세로 : ");
		double b = sc.nextDouble();

		System.out.println("면적 : " + a*b);
		System.out.println("둘레 : " + 2*(a+b));
	}
	public void example3() {
		Scanner sc = new Scanner(System.in);

		System.out.println("문자열을 입력하시오 : ");
		String str = sc.nextLine();
		
		for(int i =0; i<str.length(); i++) {
			System.out.println((i+1) + "번째 문자 : " + str.charAt(i));
		}
	}
}
```

```java
package com.example;

public class TestMain {
	public static void main(String[] args) {
		Example ex = new Example();
		
		ex.example1();
		System.out.println();
		ex.example2();
		System.out.println();
		ex.example3();
	}
}
```

### 형변환 실습

- 문제
    
    [형변환 실습]
    
    - 새 패키지 추가 : com.example2
    - 실행 테스트용 클래스 : TestMain (main() 포함 : 아래의 메소드 실행 테스트)
    - 기능 제공용 클래스 : CastingSample (각 문제별 메소드 추가함)
    
    [실습 1]
    public void printUniCode(){}
    
    - 키보드로 문자하나를 입력받아, 그 문자의 유니코드를 출력되게 함
    - 입력예 :
    문자 입력 : A
    - 출력예 :
    A is unicode : 65
    
    [실습2]
    public void calculatorScore(){}
    
    - 실수형으로 국어, 영어, 수학 세 과목의 점수를 입력받아 총점과 평균을 계산해서 출력함
    - 총점과 평균은 정수형으로 처리함
    - 입력예 :
    국어 : 90.0
    영어 : 90.0
    수학 : 90.0
    - 출력예 :
    총점 : 270
    평균 : 90

```java
package com.example2;

import java.util.Scanner;

public class CastingSample {
	public void printUniCode() {
		Scanner sc = new Scanner(System.in);
		
		System.out.println("Character input : ");
		char in = sc.next().charAt(0);
		int in_unicode = in;
		
		System.out.println(in + " is unicode : " + in_unicode);
	}
	public void calculatorScore() {
		Scanner sc = new Scanner(System.in);
		double korean, english, math;
		int total, avg;

		System.out.println("국어 : ");
		korean = sc.nextDouble();
		System.out.println("영어 : ");
		english = sc.nextDouble();
		System.out.println("수학 : ");
		math = sc.nextDouble();
		
		total = (int)(korean + english + math);
		avg = total / 3;

		System.out.println("총점 : " + total);
		System.out.println("평균 : " + avg);
		
	}
}
```

```java
package com.example2;

public class TestMain {

	public static void main(String[] args) {

		CastingSample cs = new CastingSample();
		
		System.out.println("[ 실습 1 ]");
		cs.printUniCode();
		System.out.println("---------------");
		System.out.println("[ 실습 2 ]");
		cs.calculatorScore();
	}

}
```
